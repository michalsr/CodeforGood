If your theme supports Wordpress 2.9's thumbnail feature, this plugin displays thumbnails on the admin area's manage posts and manage pages screen.

== Description ==

If your theme supports the thumbnail feature introduced in Wordpress 2.9, this plugin will display attached thumbnails on the admin area's manage posts and manage pages screen, similar to the media management page.
	
== Installation ==

Extract the zip file and just drop the contents in the
wp-content/plugins/ directory of your WordPress installation
and then activate the Plugin from Plugins page. See if you
like, what it does. There is no options page.
